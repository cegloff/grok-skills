```markdown
# grok-skills

A monorepo for reusable, high-quality Grok skills.

## Skills

- **[web-live-debug](./skills/web-live-debug/)** — Live browser debugging using remote debugging (CDP), Playwright, and multi-subagent automation. Works with any local web application.

## Usage

Each skill lives in `skills/<skill-name>/` and follows the official Grok skill format (`SKILL.md`).

## Contributing

New skills are welcome. Please open an issue first to discuss the scope.

## Platform Support

Skills in this repo aim to support:
- Windows + WSL (with mirrored networking)
- Native Linux
- macOS
```
